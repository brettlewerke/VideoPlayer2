# VideoPlayer2 Assets

This directory contains application assets including:

- `icon.png` - Application icon (512x512 recommended)
- `logo.png` - Application logo for splash screens
- `placeholder-movie.jpg` - Default movie poster placeholder
- `placeholder-show.jpg` - Default TV show poster placeholder
- Other UI assets

## Icon Requirements

For cross-platform compatibility, provide:
- `icon.png` (512x512) - Main application icon
- `icon.ico` (Windows)
- `icon.icns` (macOS)
- Various sizes for web/progressive web app if needed

## Development Notes

Placeholder assets are used during development. Production builds should include optimized, final assets.
